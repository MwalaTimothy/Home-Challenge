# Images

This directory contains visual assets for the Carenuity IoT Smart Home Challenge roadmap.

## Directory Structure

```
images/
├── level-1/          # Level 1 related images
├── level-2/          # Level 2 related images
├── level-3/          # Level 3 related images
├── level-4/          # Level 4 related images
├── level-5/          # Level 5 related images
├── level-6/          # Level 6 related images
├── diagrams/         # System and circuit diagrams
├── hardware/         # Hardware photos and pinouts
└── screenshots/      # Software screenshots
```

## Image Guidelines

### Adding Images

When contributing images:

1. **Use descriptive filenames**
   - Good: `esp32-cam-pinout.png`
   - Bad: `image1.png`

2. **Optimize file size**
   - Compress images before uploading
   - Maximum resolution: 1920x1080 for photos
   - Use appropriate format: PNG for diagrams, JPG for photos

3. **Include attribution**
   - Credit original creators
   - Note if image is original or modified
   - Include license information

### Recommended Tools

- **Image Editing:** GIMP, Paint.NET, Photoshop
- **Compression:** TinyPNG, ImageOptim
- **Diagrams:** Fritzing, KiCAD, draw.io
