## Introduction

The Carenuity IoT Smart Home Challenge is a structured program designed to guide students through progressively complex tasks in the realm of IoT and smart home devices. This challenge is targeted at beginners with basic knowledge in electronics and programming, aiming to enhance both technical skills and practical knowledge.

Each level builds on the previous one, ensuring a comprehensive learning experience that covers topics from basic soldering and simulation to advanced machine learning and image recognition.

### Target Audience

- Students interested in IoT and embedded systems
- Hobbyists wanting to build smart home devices
- Developers transitioning to hardware and IoT
- Anyone passionate about AI-assisted edge computing

### What You Will Learn

- Electronics fundamentals and soldering
- PCB design and modification
- Embedded programming with Arduino and ESP32
- API integration and application development
- 3D modeling and enclosure design
- Database integration and data logging
- Machine learning and AI model deployment
- Edge computing with computer vision

## Challenge Overview

The challenge consists of six progressive levels, each with specific learning objectives, tools, and rewards:

| Level | Focus Area | Difficulty | Reward |
|-------|-----------|------------|--------|
| 1 | Soldering & Simulation | Basic | D1-Mini / C3-Mini |
| 2 | PCB Design & GPIO Monitoring | Basic | 0.66 / 0.96 OLED Display |
| 3 | API Usage & App Development | Basic | DS18B20 / Oximeter Sensor |
| 4 | 3D Modeling & Enclosures | Advanced | 25% ChipGlobe.shop Voucher |
| 5 | Database & ML Preparation | Advanced | ESP32-CAM + Cloud Access |
| 6 | AI Edge Computing | Advanced | Hero Website + Device Stocking |

- Sensor integration basics

- GitHub account

### Level 2: PCB Design & GPIO Viewer

**Objective:** Learn PCB design modification and real-time GPIO pin monitoring

**Description:**

#### Key Skills

- KiCAD software proficiency
- Gerber file generation
- GPIO pin monitoring
- Microcontroller programming

#### Tools Required
- KiCAD software
- GPIO Viewer application
- C3-Mini microcontroller (from Level 1)
- Smartphone or tablet

#### Learning Path

**1. KiCAD Fundamentals**
- [KiCAD Getting Started](https://docs.kicad.org/master/en/getting_started_in_kicad/getting_started_in_kicad.html) - Official guide
- [KiCAD PCB Design Tutorial](https://www.youtube.com/watch?v=vaCVh2SAZY4) - Complete walkthrough

**2. PCB Design Modification**

- [PCB Design Best Practices](https://www.altium.com/documentation/altium-designer/pcb-design-best-practices) - Guidelines

**3. GPIO Viewer Setup**
- Download GPIO Viewer firmware from [Carenuity C3-Mini page](https://carenuity.com/c3-mini)
- Flash firmware to C3-Mini microcontroller
- Connect to access point (SSID: GPIO-Viewer, Password: password)
- Access interface at 192.168.4.1

**4. PCB Customization Project**
- Clone [Carenuity KiCAD PCB Repository](https://github.com/Carenuity/KiCAD-PCB)
- Modify silkscreen layer with custom logo
- Generate Gerber files for manufacturing
- Document design decisions

#### Deliverables
- Gerber files (ZIP archive) of modified PCB
- Screenshot of GPIO Viewer interface on smartphone
- Design documentation

#### Resources
- [Carenuity KiCAD Repository](https://github.com/Carenuity/KiCAD-PCB)
